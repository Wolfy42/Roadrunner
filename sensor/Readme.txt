Um die mit nodejs simulierten sensoren zu starten müssen folgende Abhängigkeiten
installiert werden:

- git clone https://github.com/joyent/node.git && cd node
- ./configure
- make
- sudo make install
- sudo ./install_npm.sh
- sudo npm install optimist




Sensor starten mit:
node sensor.js --ip 127.0.0.1 --port 4711
